LED Color Settings

VFR Flight Category:
VFR is designated by GREEN (0,255,0).
MVFR Flight Category:
MVFR is designated by BLUE (0,0,255).
IFR Flight Category:
IFR is designated by RED (255,0,0).
LIFR Flight Category:
LIFR is designated by Magenta (255,0,255).
No Weather Reported:
No Weather is typically designated by GRAY (80,80,80) or BLACK (0,0,0).
Black/Off:
Black/Off is designated by BLACK (0,0,0).
Lightning:
Lightning is typically designated by YELLOW (255,255,0).
Snow1 Reported:
Snow1 Reported is typically designated by WHITE (255,255,255).
Snow2 Reported:
Snow2 Reported is typically designated by SILVER (100,100,100).
Rain1 Reported:
Rain1 Reported is typically designated by DARK BLUE (4,0,54).
Rain2 Reported:
Rain2 Reported is typically designated by BLUE (0,0,255).
Freezing Rain1 Reported:
Freezing Rain1 Reported is typically designated by LIGHT PURPLE (199,156,219).
Freezing Rain2 Reported:
Freezing Rain2 Reported is typically designated by PURPLE (152,0,199).
Dust, Sand, Ash1 Reported:
Dust, Sand or Ash1 Reported is typically designated by BROWN (200,150,125).
Dust, Sand, Ash2 Reported:
Dust, Sand or Ash2 Reported is typically designated by DARK BROWN (100,50,0).
Fog1 Reported:
Fog1 Reported is typically designated by SILVER (80,80,80).
Fog2 Reported:
Fog2 Reported is typically designated by SILVER (80,80,80).
Home Airport Marker Color:
Home Airport Marker Color can be any color that stands out.

LCD / OLED / Rotary Settings

Number of Airports to Display:
Number of highest-wind airports to display. 5 to 10 is good.
Display an Exclusive Group of Airports:
If Yes, only airports listed in the next setting will be displayed.
List of Exclusive Airports to Display:
Subset list of airports to display on the OLED's.
Display Only Above kts:
If Yes, only airports with winds above set speed will be displayed.
LCD Display Pause:
Pause between character movements in scroll for LCD only.
Rotate OLED Display 180 Degrees:
Set to 'Yes' if OLEDs are mounted upside down.
Reverse OLED's Position Order:
Set to 'Yes' if OLEDs are wired backwards.
OLED Display Pause:
Pause between character movements in scroll for OLED only.
OLED Font Size:
Size of font for OLED display. 24 works well.
OLED Pixel Offset:
Pixel offset for OLED text display vertically.
OLED Display Wind Direction as:
Choose how to display wind direction and speed.
OLED Bold Highest Speed Airport:
If Yes, Bold the text for the airport with the highest windspeed.
OLED Add Blank Screen:
If Yes, Add a blank screen between the group of airports.
OLED Display Border:
If Yes, draw a border around the text on the display.
OLED Level Of Dimming:
Sets brightness of OLED displays if IC238 Ambient Light Sensor is NOT used.
OLED Minimum Brightness:
Set value 0-255 for the minimum brightness.
OLED Maximum Brightness:
Set value 0-255 for the maximum brightness.
OLED Invert Display:
Normal Display is white text on black background.
OLED Toggle Inverted Display:
If Yes, Toggle Inverted display between groups of airports.
OLED Scroll Display Direction:
Used on multiple OLEDs display only.
OLED Display Welcome Message:
If Yes, display a welcome message.
OLED Welcome Message:
Must be in quotes. Use '\n' for line breaks.
OLED Display Local/Zulu Time:
If Yes, Display the local and Zulu Time.
Display IP Address:
If Yes, Display the Raspberry Pi's IP address.
